----------------------------------------

📚 프로젝트 개요

총 4개의 페이지로 구성되어 있습니다.

Start 페이지: 사이트 첫 화면

Leaf 페이지: 내가 진행한 프로젝트 소개

Stem 페이지: 자기소개 페이지

Nature 페이지: 마지막 페이지

React를 사용하여 컴포넌트 기반으로 개발하였으며, 각 페이지는 독립된 컴포넌트로 관리됩니다.

----------------------------------------
